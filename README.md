# Revised Martial Equipment (RME)

A Foundry VTT module that adds revised martial equipment content for the D&D 5e system.

---

## Troubleshooting

- Manifest URL opens correctly in browser
- ZIP download link works
- GitHub repository is public
- Release tag matches the download URL
- Version number is updated in module.json

The "download" link should be the one that automatically downloads the source.zip
The "manifest" link should be the same as the raw link to the module.json

---

## Installation

1. Open **Foundry VTT**
2. Go to **Add-on Modules**
3. Click **Install Module**
4. Paste the **Manifest URL**
5. Click **Install**

Manifest URL = Raw link to module.json
